Project Description

Background:

  • We were given the market dataset provided by IEX, which relating to stock price.
  • Sentiment activities have been captured by using NLP technique on Twitter.
  • The prediction of earning estimate has been provied by ESZ and WS.

Purpose:

  • Explore the relationship between stock price and sentiment activities.
  • Track stock trend of typical company's to validate the true stock performance with its PsychSignal Sentiments.
  • Summarize the average stock return before implemented the investment portfolio strategies.

Data Understanding

  1. "estRelease" Dataset:
    • Symbols: The abbreviation of each company.
    • Release Time: The date when estimate result released.
    • Actual EPS/ESZ EPS/WS EPS
    • Actual REV/ESZ REV/WS REV
    • Beat/Miss: Actual EPS/ESZ
    • Beat/Miss: Actual EPS/WS
    • Explaination: This dataset shows the quarterly comparison between actual earnings and ESZ/WS predicted earnings across specific companies.

  2. "1Year Stock-price" Dataset:
    • Symbols: The abbreviation of each company.
    • Time duration: The date when stock price information is available.
    • High/Low/Open/Close price: The price of stocks everyday.
    • Transaction Volumn: The transaction volumn of stocks everyday.
    • Explaination: This dataset contains 1year stock price for each company.

  3. "Psychsignal" Dataset:
    • Symbols: The abbreviation of each company.
    • Time duration: The date when psychsignal information is availble.
    • Bullish/Bearish intensity: The possible expectation if stocks have positve tendency.
    • Sentiment score: The score that reflects sentiment conditon.
    • Explaination: This dataset contains a series of score and ratio that shows how stock price may goes up and goes down.

  4. "SEC-MASTER" Dataset:
    • Symbols: The abbreviation of each company.
    • Sector/Industry: The industry or sector that each company is belonging to
    • Explaination:This dataset contains the introduction of each company, like industry, sector, marketcap...

Project Objectives

How Does Extremely PsychSignal Sentiments Affects the stocks return?

How Much Money Will Investors Win or Loss When PsychSignals Appeared?
